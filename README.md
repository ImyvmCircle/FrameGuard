# FrameGuard
Protect picture frames and paintings from other players with Minecraft game.

# Installation method
Please put jar in the plugins folder.

# Useage
```
/fglock   lock
/fgunlock unlock
/fginfo   information
```

# Permission 
```
frameguard.fglock
frameguard.fgunlock
frameguard.fginfo
```
